# graf-appColorGithub
una aplicación interactiva tipo “RGB Color Picker”, diseñada para permitir al usuario seleccionar y ajustar colores mediante el modelo de color RGB (Rojo, Verde y Azul), mostrando también su equivalente en código hexadecimal (HEX). La interfaz es moderna, limpia y fácil de usar, enfocada en la experiencia del usuario.`
